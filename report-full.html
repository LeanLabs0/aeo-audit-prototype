Snippet 1 of 5 FAQPage schema · /pricing Critical

Why this matters: Lean Labs' pricing page already answers buyer questions in prose. Wrapping each Q&A in FAQPage JSON-LD turns those answers into AI-extractable blocks. Single biggest AEO win on the site.

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "How much does Lean Labs cost?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Lean Labs engagements start at $5,000 for one-time AEO Core fixes and scale to $50,000 for full 90-day AEO Domination programs. Pricing depends on scope, deal value, and timeline."
      }
    },
    {
      "@type": "Question",
      "name": "What's included in the AEO Core package?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Schema markup deployment across top pages, FAQ schema generation, and a 2-week implementation cycle. Delivered as a one-time engagement."
      }
    }
  ]
}
</script>
Snippet 2 of 5 Organization sameAs · site-wide High

Why this matters: AI engines disambiguate brand mentions through canonical profile links. Adding LinkedIn, Crunchbase, and G2 to your Organization schema gives them an anchor.

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Lean Labs",
  "url": "https://www.lean-labs.com",
  "logo": "https://www.lean-labs.com/logo.png",
  "sameAs": [
    "https://www.linkedin.com/company/lean-labs-llc/",
    "https://www.crunchbase.com/organization/lean-labs",
    "https://www.g2.com/products/lean-labs/reviews"
  ]
}
</script>
Snippet 3 of 5 Article schema w/ author + dateModified · blog posts High

Why this matters: AI engines weight author credentialing + content freshness heavily. Adding dateModified to existing Article schema is the smallest possible change with the biggest freshness lift.

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "BlogPosting",
  "headline": "How Long Does It Take to Get Results from Inbound Marketing?",
  "datePublished": "2024-08-15T00:00:00-05:00",
  "dateModified": "2026-05-13T00:00:00-05:00",
  "author": {
    "@type": "Person",
    "name": "Kevin Barber",
    "url": "https://www.linkedin.com/in/kevinrayebarber/"
  }
}
</script>
